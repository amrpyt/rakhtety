# Quick Plan: Simulate Full Office E2E Flows

## Goal

Run a real office-style rehearsal for the app using fresh test accounts:

- Admin flow: login, manage employees, create/open client, create workflow, check report/dashboard.
- Employee flow: login, open clients, update workflow steps, record payment or identify blocker, confirm restricted admin actions are hidden/blocked.

## Steps

1. Create fresh test admin and employee accounts.
2. Record the new test credentials in `AGENTS.md`.
3. Use Browser Use to log in as admin and verify admin pages/actions.
4. Use Browser Use to create or open a test client and prepare workflow data.
5. Use Browser Use to log in as employee and verify allowed/restricted behavior.
6. Capture pass/fail notes in a GSD summary.
7. Run checks and commit only this edit round.
