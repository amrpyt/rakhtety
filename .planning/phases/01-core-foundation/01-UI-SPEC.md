## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f7f6f2` | Page background (`--color-bg`) |
| Secondary (30%) | `#f9f8f5` | Cards, sidebar (`--color-surface`) |
| Accent (10%) | `#01696f` | Primary actions, active nav, focus rings |
| Destructive | `#b91c1c` | Destructive actions, error states |

**Accent reserved for:**
- Primary button background
- Active nav item background + text
- Focus ring outline (`2px solid var(--color-primary)`)
- Form input focus border

**Not accent:**
- Success indicators (use `--color-success` `#437a22`)
- Warning indicators (use `--color-warning` `#964219`)
- Financial amounts (fees use warning, profit use success)

### Workflow UI (Phase 1 — Read-Only Skeleton)

**Device License (5 steps):**
1. بيان الصلاحية
2. تقديم المجمعة العشرية للإسكان المميز
3. تقديم الملف
4. دفع إذن الرخصة وشراء عقد مخلفات
5. استلام الرخصة

**Excavation Permit (5 steps):**
1. تقديم واستلام شهادة الإشراف
2. تقديم واستلام التأمينات
3. التقديم على العداد الإنشائي
4. تقديم ودفع واستلام تصريح الحفر
5. تصريح التعدين

**Each step displays:**
- Step name (Arabic)
- Status badge (pending/in_progress/completed/blocked)
- Government fees (رسوم) — shown in `--color-warning` (`#964219`)
- Office profit (أتعاب) — shown in `--color-success` (`#437a22`)
- Assigned employee name
- Completion date (if completed)

**Dependency indicator:** Excavation Permit tab visually locked (grayed) until Device License workflow is COMPLETED.

## Phase 1 UI Scope
